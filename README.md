## 使用方法

#### オプション一覧
以下、例
| オプション | 説明 |
| :--- | :--- |
| -i    | 入力画像のパス(必須) |
| -p    | 特徴点データのパス(必須) |
| -o    | 出力画像のパス |
| -c    | 色の設定 |
| -s    | 形状の設定 |
| -r    | 半径の設定 |
| --shift | 画像の中心を原点とする場合 |

#### 実行例
以下、例
```bash
python3 main.py -i ./image/sample.png -p ./points_data/sample.dat
```
